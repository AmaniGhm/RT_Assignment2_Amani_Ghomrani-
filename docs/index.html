Action client node¶

Subscribes to:
/odom
Publishes to:
/robot_state
class action_client_node.ActionClientNode[source]¶

A ROS node that acts as an action client for sending goals and receiving feedback.

This node subscribes to the /odom topic to receive odometry messages and publishes the robot state to the /robot_state topic. It also communicates with an action server to send goals and cancel them if needed.

This function processes the incoming odometry messages and publishes the robot state.

Position node¶

Subscribes to:
/robot_state
Publishes to:
None
Services:
distance_velocity_from_target
Description:
This ROS node subscribes to the /robot_state topic to receive updates on the robot’s position and velocity. It calculates the distance from a target position and the average speed of the robot, printing this information periodically. Additionally, it provides a service to retrieve the current distance and average speed.
position_node.dist_velocity_callbk(rsp)[source]¶

Service callback function to handle requests for the robot’s distance from the target and its average speed.

Parameters:rsp (GetDistSpeedRequest) – Request from the GetDistSpeed service.
Returns:Response containing the distance and average speed of the robot.
Return type:GetDistSpeedResponse
position_node.print_robot_info(msg)[source]¶

Callback function for the /robot_state topic subscriber. It calculates the distance from the target position and the average speed of the robot, then prints this information periodically based on the frequency parameter.

Parameters:msg (RobotState) – Message containing the current state of the robot, including position and velocity.

Target service node¶

Subscribes to:
/reaching_goal/goal (message type: PlanningActionGoal)
Services:
get_last_target (service type: Getlasttarget) - Returns the latest target coordinates (target_x, target_y).
Description:
This ROS node retrieves the last target coordinates received on the /reaching_goal/goal topic and provides them as a service response. The service is named get_last_target and returns a message of type GetlasttargetResponse containing the target_x and target_y values.
